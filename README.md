# 🕊️ Generador de Grupos · Comunidad

Herramienta web para generar grupos aleatorios respetando matrimonios y unidades inseparables, con interfaz visual y drag & drop.

## ✨ Funcionalidades

- **Generación automática** de grupos de 4–6 personas
- **Matrimonios y Seminaristas** siempre juntos en el mismo grupo
- **Responsables automáticos** — asigna responsable preferente si está en el grupo, o uno aleatorio si no
- **Drag & drop** — arrastra personas entre grupos para ajustar manualmente
- **Gestión de personas** — añade, elimina y renombra desde la propia app
- **Modo claro / oscuro**
- **Responsive** — funciona en móvil, tablet y ordenador

## 🚀 Uso

Accede directamente desde el navegador:

